# Skill Template

This template describes the expected SKILL.md structure and sections.

```markdown
---
name: {skill_name}
description: {description}
---

# {Skill Title}

## Overview

[1-2 sentences explaining what this skill enables]

## Instructions

[Step-by-step instructions or guidelines]

## Examples

[Concrete examples showing how to use the skill]
```

Guidance:
- Name: lowercase letters, numbers, hyphens only, <= 64 chars.
- Description: include “what it does” + “when to use”.
- Keep SKILL.md concise; move large details to references/ when needed.
